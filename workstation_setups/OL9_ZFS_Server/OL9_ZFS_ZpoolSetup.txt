5Ô∏è‚É£ Set useful dataset defaults (highly recommended)

Instead of using the pool root directly, create datasets:

$ sudo zfs set compression=lz4 tank
$ sudo zfs set atime=off tank
$ sudo zfs set xattr=sa tank
$ sudo zfs set acltype=posixacl tank
 
$ sudo zfs set compression=lz4 tank/backups
$ sudo zfs set atime=off tank/backups
$ sudo zfs set xattr=sa tank/backups
$ sudo zfs set acltype=posixacl tank/backups


Then create datasets as needed:

$ sudo zfs create tank/data
$ sudo zfs create tank/backups


Check:

$ zfs list
